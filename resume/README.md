# Vaishali Sharma — ATS-Friendly Resume

Fresher / Frontend Developer resume optimized for Applicant Tracking Systems (ATS).

## Files

| File | Use |
|------|-----|
| `Vaishali_Sharma_ATS_Resume.pdf` | Ready-to-upload 1-page PDF |
| `Vaishali_Sharma_ATS_Resume.html` | Open in a browser → Print → Save as PDF |
| `Vaishali_Sharma_ATS_Resume.md` | Copy into Microsoft Word / Google Docs → export as `.docx` or PDF |
| `generate_pdf.py` | Regenerates the PDF (`python3 generate_pdf.py`) |

## How to export from HTML

1. Open `Vaishali_Sharma_ATS_Resume.html` in Chrome or Edge.
2. Press **Ctrl+P** (Windows) / **Cmd+P** (Mac).
3. Destination: **Save as PDF**.
4. Margins: Default or Minimum.
5. Save as `Vaishali_Sharma_Resume.pdf`.

## ATS design choices

- Single-column layout (no multi-column tables)
- Standard section headings ATS parsers recognize
- Plain Helvetica/Arial text (no icons, graphics, or text boxes)
- Keywords for Frontend Developer roles: React.js, JavaScript, HTML5, CSS3, Tailwind CSS, Flask, Django, Python
- Consistent date format (MM/YYYY)
- Contact details as plain text (not in headers/footers)
- Fits on one page
